# Instagram Growth Audit Engine

Production-ready tool for deep Instagram profile analysis.

## Setup Instructions

### Backend (Flask)
1. Navigate to `/backend`
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with `OPENAI_API_KEY=your_key_here`
4. Run: `python app.py`

### Frontend (React)
1. Navigate to `/frontend`
2. Install dependencies: `npm install`
3. Run: `npm run dev`

## Core Features
1. **Automated Scraping**: Fetch 20-30 posts via Instaloader.
2. **AI Audit Engine**: GPT-4 driven 4-module growth strategy.
3. **Data Visualization**: Heatmaps and engagement trends via Recharts.
4. **Export**: Generation of PDF reports.
